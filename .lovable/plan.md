## เพิ่มฟิลเตอร์พนักงาน และ ขั้นตอน ในหน้าแดชบอร์ด

ปัจจุบันหน้าแดชบอร์ดมีเฉพาะฟิลเตอร์ "หมวดหมู่งาน" จะเพิ่มอีก 2 ตัวให้ผู้ใช้กรองข้อมูลได้ละเอียดขึ้น

### สิ่งที่จะทำ (ไฟล์เดียว: `src/routes/_protected.dashboard.tsx`)

1. **โหลดข้อมูลเพิ่ม**: ดึงรายชื่อพนักงาน (`employees`) และขั้นตอน (`steps`) ที่ active มาเก็บ state
2. **เพิ่ม state ฟิลเตอร์ 2 ตัว**:
   - `employeeFilter` (default: `"all"`)
   - `stepFilter` (default: `"all"`)
3. **อัปเดต `scopedLogs`**: ขยาย logic ให้กรองตามหมวดหมู่ + พนักงาน + ขั้นตอน พร้อมกัน — สถิติ/กราฟ/อันดับ/MoM/over-standard ทั้งหมดจะอัปเดตอัตโนมัติ เพราะอ้างอิง `scopedLogs`
4. **UI ใน toolbar ด้านบน**: เพิ่ม `<Select>` 2 ตัวข้างๆ ฟิลเตอร์หมวดหมู่
   - "พนักงานทั้งหมด" + รายชื่อ
   - "ทุกขั้นตอน" + รายการขั้นตอน
   - บนจอแคบจะ wrap ลงบรรทัดใหม่ (ใช้ `flex-wrap` ที่มีอยู่แล้ว)
5. **ปุ่ม "ล้างฟิลเตอร์"**: แสดงเมื่อมีฟิลเตอร์อย่างน้อยหนึ่งตัวถูกตั้ง เพื่อรีเซ็ตทั้ง 3 กลับเป็น "all"

### หมายเหตุ
- ไม่แตะ schema/ฐานข้อมูล
- ไม่กระทบส่วนอื่นของแอป
